 Fusion Knowledge Base

Introduction
============

The Fusion Knowledge Base is a growing collection of articles dedicated to
documenting features, common tasks and pitfalls in Fusion.

If there are any topics that should be documented but aren't, please contact
`Fusion support <fusion-support_>`_ with your suggestion.

.. _fusion-support: support@fusionapp.com

Using the Knowledge Base
------------------------

Available articles are listed in the sidebar to the left and grouped under their
respective topics, as well as below.

The Knowledge Base can be searched from the sidebar simply by entering some
search criteria and pressing :kbd:`Enter`.
